# excel-sales-dashboard

                          # Dynamic Excel Sales Dashboard using VBA and Macros #

## Project Overview
This project is an interactive sales dashboard developed using Microsoft Excel, VBA, and Macros. The dashboard analyzes sales executive performance, target achievement, and regional sales insights with automated dashboard navigation features.

---

## Key Features
- Interactive Dashboard Navigation Buttons
- VBA and Macro Automation
- Dynamic City-wise Dashboard Switching
- Sales Performance Tracking
- Target Achievement Analysis
- Automated Dashboard Interaction

---

## Dashboard Functionalities
### Dashboard 1
- Sales Executive-wise Total Sales Analysis
- Horizontal Bar Chart Visualization

### Dashboard 2
- Sales Comparison Across Executives
- Target Performance Metrics

### Dashboard 3
- Target Hit Percentage Analysis
- Pie Chart Representation

### Dashboard 4
- Away From Target Percentage Analysis
- Trend Line Visualization

---

## Technologies Used
- Microsoft Excel
- VBA (Visual Basic for Applications)
- Macros
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

## Advanced Features Implemented
- Automated button-based dashboard navigation using VBA
- Dynamic switching between multiple dashboards
- Interactive city filters
- Macro-enabled dashboard automation
- Real-time KPI updates

---

## KPIs Included
- Total Sales
- Target Hit %
- Away From Target %
- Sales Executive Performance

---

## Insights Generated
- Identified top-performing sales executives
- Compared target achievement percentages
- Analyzed city-wise sales performance
- Tracked executives below target achievement

---

## Project Outcome
Enhanced Excel automation, dashboard development, VBA programming, and business analytics skills through interactive reporting and macro-based automation.
